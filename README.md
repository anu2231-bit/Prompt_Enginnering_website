** Startup Landing Page**

A modern, vibrant, responsive startup landing page built using HTML + TailwindCSS.
This project includes a complete UI layout with multiple sections designed in a premium purple–blue–pink gradient theme.

** Features**

Fully responsive UI (mobile-first)

Modern gradient theme & startup-style visuals

Clean Navigation Bar with mobile menu

Eye-catching Hero Section

Services / Features Section with animated cards

Contact / Lead Form with glassmorphic design

Footer Section with consistent theme

Smooth spacing, rounded cards, soft shadows

Uses free royalty-free images from Unsplash/Pexels

Built entirely with HTML + TailwindCSS CDN
